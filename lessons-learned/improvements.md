# Improvements and Future Work

This document tracks improvements identified during the project.

- Raspberry Pi 5
- External SSD via M.2 HAT

These will unlock further work.

Future work includes:

- Automated backups
- Off-device storage
- Restore testing

Data safety will improve incrementally.

## Documentation

Additional documentation planned:

- Bootstrap automation
- Disaster recovery walkthrough
- Service upgrade procedures

Documentation remains a priority.

## What will not be rushed

- HA control plane

- Complex networking

Complexity will only be added when justified.
